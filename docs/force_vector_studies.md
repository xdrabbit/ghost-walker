# Force Vector Studies

## Goal

Find the best cable direction for assisting hip flex.

## Candidate A: front pull
Pros:
- strong lift

Cons:
- awkward mounting
- torso pull risk

## Candidate B: lateral/anterior (recommended)
Pros:
- easy routing
- tunable
- balanced lift + forward assist

Cons:
- slight rotation possible

## Candidate C: rear redirect
Pros:
- clean packaging

Cons:
- more friction
- more complexity

## Recommendation

Start with lateral/anterior.

It gives the best balance of:
- simplicity
- adjustability
- effectiveness
